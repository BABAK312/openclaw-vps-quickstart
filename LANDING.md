## Что такое OpenClaw?

OpenClaw — это AI-агент (как Claude Code или Cursor), но работает на твоём сервере и управляется через Telegram. Пишешь голосовое — агент делает: пишет код, создаёт сайты, исправляет ошибки, деплоит на хостинг.

**Главное отличие:** Не нужно сидеть за компьютером. Агент работает на VPS, а ты управляешь им с телефона.

## Что делает скрипт (автоматически)

1. **SSH-ключи** — Генерир��ет безопасный ключ ED25519 (если нет)
2. **Подключение** — Соединяется с сервером (спросит пароль один раз)
3. **Пользователь** — Создаёт отдельного юзера `openclaw` (не root!)
4. **Обновление** — Обновляет Ubuntu до актуальной версии
5. **Hardening** — Отключает вход по паролю, только ключи
6. **Установка** — Скачивает и ставит OpenClaw
7. **Gateway** — Настраивает веб-панель
8. **Токены** — Генерирует токен авторизации
9. **Права** — Выставляет правильные permissions
10. **Проверка** — Запускает диагностику

**Всё автоматически. Ты вводишь только IP сервера.**

## Как установить

### macOS / Linux

Открой терминал и введи:

```bash
curl -fsSL https://raw.githubusercontent.com/BABAK312/openclaw-vps-quickstart/v1.0.31/install.sh | bash -s -- --host ТВОЙ_IP
```

### Windows (WSL2)

```powershell
wsl --install -d Ubuntu-24.04
wsl -d Ubuntu-24.04 -- bash -lc 'curl -fsSL https://raw.githubusercontent.com/BABAK312/openclaw-vps-quickstart/v1.0.31/install.sh | bash -s -- --host ТВОЙ_IP'
```

## После установки

### Шаг 1: Создай туннель

На своём компьютере введи:

```bash
ssh -i ~/.ssh/openclaw_vps_ed25519 -N -L 18789:127.0.0.1:18789 openclaw@ТВОЙ_IP
```

### Шаг 2: Открой браузер

Перейди: http://127.0.0.1:18789

### Шаг 3: Введи токен

Скрипт покажет токен в конце установки. Вставь его в браузере.

### Шаг 4: Подключи AI и Telegram

На сервере введи:

```bash
openclaw onboard
```

Следуй инструкциям — подключишь модель и Telegram.

---

## Почему это безопасно

- **Gateway на loopback** — панель не видна из интернета
- **SSH по ключам** — пароли отключены
- **Отдельный пользователь** — OpenClaw не работает от root
- **Нет allowlist/proxy** — никаких ограничений на запросы
- **Правильные permissions** — все файлы защищены

### Технические вопросы

**Q: Какая версия Ubuntu нужна?**
A: Ubuntu 24.04 (рекомендуется) или 22.04.

**Q: Что если что-то сломалось?**
A: Запусти `./verify.sh --host IP --repair` — скрипт починит автоматически.

**Q: Как подключиться к серверу потом?**
A: `./scripts/connect.sh --host IP` или `ssh openclaw@IP`

**Q: Где взять токен авторизации?**
A: `./scripts/get-token.sh --host IP`

**Q: Ка�� обновить OpenClaw?**
A: На сервере: `openclaw update`

---

### Вопросы безопасности

**Q: Это безопасно?**
A: Да. SSH по ключам, gateway на loopback, отдельный пользователь.

**Q: Мой токен украдут?**
A: Нет. Токен хранится на твоём сервере, не передаётся третьим лицам.

**Q: Данные утечут?**
A: Всё на твоём VPS. OpenClaw не хранит данные на своих серверах.
